**Source:** [Introducing GPT-5.4 — OpenAI](https://openai.com/index/introducing-gpt-5-4/) | [GPT-5.4 launches — TechCrunch](https://techcrunch.com/2026/03/05/openai-launches-gpt-5-4-with-pro-and-thinking-versions/) | [GPT-5.4 benchmarks — OfficeChai](https://officechai.com/ai/gpt-5-4-benchmarks/) | [GPT-5.4 safety — HelpNetSecurity](https://www.helpnetsecurity.com/2026/03/06/openai-chatgpt-gpt%E2%80%915-4-model-release/)

**What Changed:**
- **Three model variants released March 5**: GPT-5.4 (standard), GPT-5.4 Thinking (with "extreme" reasoning mode), GPT-5.4 Pro (maximum performance).
- **Native computer use**: First OpenAI model with built-in desktop/browser control via Playwright and mouse/keyboard from screenshots. 75.0% on OSWorld-Verified — surpassing human performance (72.4%).
- **1M token context window**: Largest context window from OpenAI. Long-context surcharge doubles input price above 272K tokens.
- **33% fewer hallucinations**: Individual claims 33% less likely to be false vs GPT-5.2. Full responses 18% less likely to contain errors.
- **GDPval 83%**: Matched or exceeded industry professionals in 83% of comparisons across 44 occupations.
- **Tool Search**: New API feature lets GPT-5.4 receive lightweight tool lists and look up full definitions on demand, reducing token usage by 47%.
- **Pricing**: $2.50/$15 per MTok (standard). $30/$? per MTok (Pro). Significantly cheaper than Claude Opus 4.6 ($15/$75).
- **Safety**: Safest OpenAI reasoning model. More jailbreak-resistant. Reasoning transparency (cannot hide reasoning from monitors).
- **Lockdown Mode**: Deterministic anti-prompt-injection for Enterprise/Healthcare. Disables web browsing, limits apps to admin-approved.

**Impact on You:**
- *Price disruption*: At $2.50/$15, GPT-5.4 is 6x cheaper on input and 5x cheaper on output than Claude Opus 4.6. For high-volume clinical text processing (note summarization, report generation), this changes the cost calculus dramatically.
- *Computer use for clinical workflow automation*: Agents that can navigate EHR screens, fill forms, and verify data across applications. Evaluate for prior auth automation, chart review, and cross-system data entry.
- *Lockdown Mode for healthcare*: Deterministic prompt injection protection is exactly what healthcare AI deployments need. Evaluate alongside Claude's HIPAA BAA offering.
- *Context window parity*: Both Claude and GPT-5.4 now offer 1M context. OpenAI is no longer at a context disadvantage.

**Disruption Score: 10/10** — Evaluate GPT-5.4 immediately. Run head-to-head against Claude Opus 4.6 for your top 3 clinical use cases. The pricing difference alone warrants re-evaluation of model routing strategy.
